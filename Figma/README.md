## 快捷鍵 Windows

- `alt` : 檢查物體間距
- `ctrl + D` : 複製相同物件
- `ctrl + 上下左右` : 微調物體大小 1px
- `ctrl + shift + 上下左右` : 微調物體大小 10px
- `ctrl + R` : 圖層命名
- `enter` : 進入下一層元素
- `shift + enter` : 回到上一層元素
- `tab` : 切換圖層中下個元素
- `shift + tab` : 回到圖層中上個元素
- `ctrl + [ or ]` : 將元件移動到上/下一層
- `N` : 切換與瀏覽 Frame 
- `alt + L` : 收起所有圖層 
- `ctrl + shift + H` : 隱藏/顯示圖層 
- `shift + H or V` : 垂直/水平翻轉
- `ctrl + 1-9` :  切換檔案
- `shift + 0-2` :  切換瀏覽視角
- `ctrl + \` :  簡報模式(關閉左右選單)
- `ctrl + /` :  呼叫主菜單
- `alt + 1,2,8,9,0` :  切換面板
- `ctrl + K` :  新增連結
