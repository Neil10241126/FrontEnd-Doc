# GSAP 學習文件
---

## 開始安裝
此處查看[完整文件](https://greensock.com/docs/)，包含[影片安裝指南](https://greensock.com/docs/v3/Installation)
### CDN
```js
<script src="https://cdn.jsdelivr.net/npm/gsap@3.12/dist/gsap.min.js"></script>
```

### NPM
```
npm install gsap
```

### 匯入 GSAP 模組
```js
// 最基礎的 GSAP 模組
import { gsap } from "gsap";
```
